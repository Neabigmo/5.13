# Submission checklist for Machine Learning

Before upload, verify the latest official Springer instructions.

## Manuscript

- [ ] Title matches manuscript and submission portal.
- [ ] Abstract is concise and not overclaimed.
- [ ] All theorem/proposition references compile correctly.
- [ ] All figures are cited in order.
- [ ] No figure appears accidentally after references.
- [ ] All tables are generated from experiment outputs.
- [ ] References are complete and real.
- [ ] No internal notes such as `current draft`, `v0.1`, `TODO`, `Figure 8` remain.

## Source package

- [ ] `main.tex`
- [ ] `sn-jnl.cls` and bibliography style files, if license allows bundling
- [ ] `refs.bib`
- [ ] all `sections/*.tex`
- [ ] all figures as PDF/PNG
- [ ] all generated tables
- [ ] declarations file

## Supplementary material

- [ ] README with reproduction commands
- [ ] environment file
- [ ] raw result manifest or link to archive
- [ ] code repository link
- [ ] version/commit hash

## Editorial material

- [ ] cover letter
- [ ] competing interests statement
- [ ] funding statement
- [ ] data availability statement
- [ ] code availability statement
- [ ] author contributions
- [ ] AI-use disclosure if required/applicable
- [ ] suggested reviewers, if requested
