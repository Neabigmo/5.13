# Reproducibility checklist

- [ ] One-command experiment runner exists.
- [ ] All random seeds are recorded.
- [ ] All generated raw files have checksums.
- [ ] All summary tables are generated from raw files.
- [ ] All figures are generated from saved summaries.
- [ ] The code version is recorded by git commit hash.
- [ ] Dependency versions are saved.
- [ ] Theoretical witness constructors are unit-tested.
- [ ] Insert-one semantics are unit-tested.
- [ ] Odd-k diagnostic precision/recall is unit-tested.
