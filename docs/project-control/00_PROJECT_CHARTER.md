# Project Charter

## Goal

Develop a high-quality research project and paper on stability calculus for local learning rules, centered on exact delete-one, replace-one, and leave-one-out characterizations for k-nearest neighbors.

## Working Title

**A Stability Calculus for Local Learning Rules: Exact Delete-One, Replace-One, and Leave-One-Out Characterizations for k-Nearest Neighbors**

## Roles

- Codex owns definitions, theorem truth, proof audit, literature judgment, final narrative, and acceptance decisions.
- Claude Code owns bounded implementation, enumeration, plotting, LaTeX maintenance, and mechanical formatting tasks.

## Quality Bar

- Highest quality by default.
- No skipped tests.
- No weakened proof goals without explicit user approval.
- No unverified novelty claims.
- No computational evidence stated as mathematical proof.

## Forbidden

- Do not commit secrets.
- Do not let Claude Code redefine concepts.
- Do not blur LOO, delete-one, and replace-one stability.
- Do not ignore tie-breaking, duplicate points, or conflicting labels.
