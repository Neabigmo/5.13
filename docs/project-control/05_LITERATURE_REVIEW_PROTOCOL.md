# Literature Review Protocol

Codex owns literature search, screening, novelty judgment, and final citation decisions. Claude Code may format supplied entries but must not decide novelty.

## Priority Topics

- Bousquet-Elisseeff stability.
- Rogers-Wagner / Devroye-Wagner deleted estimates and local rules.
- Kearns-Ron algorithmic stability and LOO.
- Cover-Hart / Stone k-NN consistency.
- Mukherjee-Niyogi-Poggio-Rifkin LOO stability and consistency.
- k-NN LOOCV work.
- Conformal prediction references involving LOO or replace-one stability.

## Paper Note Template

Use one note per paper with citation, result summary, stability notions, relation to k-NN, relation to delete-one / replace-one / LOO, novelty risk, and must-cite status.
