\section{Additional Figures}
\label{app:additional-figures}

The submitted manuscript uses a deliberately compact figure set. Data figures
are generated from the current experiment run, while mechanism and hierarchy
figures are editable vector schematics generated by the same figure pipeline.
The source script is \texttt{experiments/nature\_figures.py}; each run records
the rendered \texttt{.svg}, \texttt{.pdf}, and \texttt{.png} assets under
\texttt{outputs/runs/ml\_submission/<timestamp>/rendered/figures/}. The
top-level \texttt{outputs/figures/} directory is only a compile mirror of the
same generated assets.
