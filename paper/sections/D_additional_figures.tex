\section{Additional Figures}
\label{app:additional-figures}

The current submission draft keeps the figure set concentrated in the main
text: perturbation semantics, margin crossing, synthetic evidence, tabular
evidence, and the stability hierarchy.  The full figure-generation record,
including editable SVG sources and raster previews, is produced by
\texttt{experiments/nature\_figures.py} and stored with the corresponding
run manifest.
